> **Q:** 写出下列常量在声明中使用的数据类型和在 printf() 中对应的转换说明：

|     常量      | 类型 | 转换说明（%转换字符） |
|:-----------:|:--:|:-----------:|
|    `12`     |    |             |
|    `0X3`    |    |             |
|    `'C'`    |    |             |
|  `2.34E07`  |    |             |
|  `'\040'`   |    |             |
|    `7.0`    |    |             |
|    `6L`     |    |             |
|   `6.0f`    |    |             |
| `0x5.b6p12` |    |             |


<details>
<summary>A:</summary>

|     常量      |        类型         | 转换说明（%转换字符） |
|:-----------:|:-----------------:|:-----------:|
|    `12`     |       `int`       |    `%d`     |
|    `0X3`    |  `unsigned int`   |    `%#X`    |
|    `'C'`    | `char`（实际上 `int`） |    `%c`     |
|  `2.34E07`  |     `double`      |    `%e`     |
|  `'\040'`   | `char`（实际上 `int`） |    `%c`     |
|    `7.0`    |     `double`      |    `%f`     |
|    `6L`     |      `long`       |    `%ld`    |
|   `6.0f`    |      `float`      |    `%f`     |
| `0x5.b6p12` |     `double`      |    `%a`     |
</details>
